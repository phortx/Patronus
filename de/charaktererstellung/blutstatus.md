# Blutstatus

## Reinblütig
  * `+2` Magie
  * `-1` Vitalität


## Halbblütig
  * `+1` Magie


## Muggelgeboren
  * `+1` Wahrnehmen
  * `-1` Magie
  * `+1` Stärke
  * `+1` Vitalität
  * `-1` Fliegen

 
## Squib / Muggle
  * Kann keine Zauber wirken, Magie immer 0
  * `+2` Vitalität
  * `+2` Stärke
  * `+2` Akrobatik
  * `+1` Redekunst
  * `+1` Wahrnehmen
